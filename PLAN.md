## 추가해야 할 폴트 로직

### 1. 쉐이크 폴트 (BWF 9.1.2 / 9.1.8)
- **규정**: 라켓 헤드 백스윙 완료 후 앞으로만 움직여야 함. 앞뒤로 흔들면 폴트
- **감지 방법**: 손목 x좌표의 방향 전환 횟수 카운트 (서브 준비~임팩트 구간)
- **필요 영상**: 쉐이크 동작 서브 20개 이상

### 2. 발 폴트 (BWF 9.1.4)
- **규정**: 서브 시작~임팩트까지 양발이 지면에 정지 상태 유지
- **감지 방법**: 발목 keypoint의 y좌표 변화량 추적 (임계값 초과 시 폴트)
- **필요 영상**: 발 끄는 서브 / 발 들리는 서브 각각 20개

### 3. 딜레이 폴트 (BWF 9.1.1 / 9.1.2)
- **규정**: 백스윙 완료 후 지나친 멈춤(undue delay) 금지
- **감지 방법**: 백스윙 완료 시점(손목 최고점) ~ 임팩트까지 시간 측정
- **필요 영상**: 딜레이 있는 서브 20개, 정상 서브 50개 (기준 시간 산정용)

### 4. 셔틀콕 높이 폴트 (BWF 9.1.6 보완)
- **규정**: 셔틀콕 자체가 허리 아래에 있어야 함 (손목 위치로 근사하는 현재 방식 보완)
- **감지 방법**: YOLOv8 커스텀 모델로 셔틀콕 직접 검출 필요
- **필요 영상**: 라벨링용 서브 영상 100개 이상 (Roboflow 라벨링)

## 촬영 세팅 (중요)
```
카메라 위치: 선수 정측면 90도, 허리 높이
거리: 전신(머리~발) + 좌우 여유 보이는 거리
프레임레이트: 60fps (임팩트 순간 정밀도)
해상도: 1080p 이상
배경: 단색 배경 or 코트 라인 보이는 곳
```

## 내일 개발 순서

1. **쉐이크 감지 로직** 추가 (손목 x방향 전환 분석)
2. **발 폴트 감지** 추가 (발목 좌표 변화량)
3. **딜레이 측정** 추가 (백스윙~임팩트 시간)
4. 수집한 영상으로 **임계값 튜닝**
5. 셔틀콕 검출용 **커스텀 YOLO 모델** 학습 (Roboflow)
