# 배드민턴 서브 폴트 검출 — 개발 플랜 v5

## 개발 단계 로드맵

```
1단계: 폴트 로직 설계 완료 ✅
     ↓
2단계: 커스텀 YOLO 학습 (셔틀콕 + 라켓헤드 + 코트라인) ← 현재
     ↓
3단계: 라이브 검출기
```

| **빽지체 폴트** | 9.1.8 | 서브 시작(9.2)~전달(9.3)까지 라켓이 계속 앞으로 이어져야 함. 뒤로 빼면 폴트 | 라켓헤드 궤적 |
| **오버헤드 폴트** | 9.1.7 | 타구 순간 라켓헤드가 아래를 향하지 않음 + 셔틀이 허리/1.10m 위 | 라켓헤드 + 셔틀콕 |

### ❌ 코트라인 인식 필요

|------|-----|----------------|------------|
| **선밟기 폴트** | 9.1.4 | 서브 시작 시 발이 코트 라인 위 또는 바깥 | 코트 라인 좌표 |

## 각 폴트 상세 판별 로직

### 웨이스트 폴트 (9.1.6)
```
판별 시점: 임팩트 프레임
조건: shuttle_y > waist_y (셔틀콕이 허리선 아래)
허리선: 어깨~엉덩이 사이 60% (갈비뼈 최하단 추정)
```

### 높이 폴트 (9.1.6.2)
```
판별 시점: 임팩트 프레임
조건: shuttle_y > height_threshold_y (1.10m 기준선 아래)
1.10m선: 신체 비율로 픽셀 변환 (1회 보정 후 고정)
```

### 샤프트/오버헤드 폴트 (9.1.7)
```
판별 시점: 임팩트 프레임
조건: racket_head_y < wrist_y (헤드가 손목보다 위 = 위를 향함)
```

### 쉐이크 폴트 (9.1.8)
```
판별 시점: 서브 시작(serve_start) 프레임
조건: 서브 시작 직전 라켓헤드의 좌우(x) 속도 > 임계값
     = 서브 시작 시 라켓이 멈추지 않고 흔들리고 있음
```

### 빽지체 폴트 (9.1.8)
```
판별 시점: 서브 시작(serve_start) ~ 임팩트 전 구간
조건: 라켓헤드의 x 이동 방향이 뒤로 전환되는 순간 감지
     = 앞으로 가다가 뒤로 빠지면 폴트
주의: 백스윙(서브 전 뒤로 빼는 동작) 끝난 후부터 체크
```

### 선밟기 폴트 (9.1.4)
```
판별 시점: 서브 시작(serve_start) 프레임
조건: ankle_y >= service_line_y (발목이 서비스 라인에 닿거나 넘음)
서비스 라인: 코트 라인 자동 감지 or 수동 설정
```

### 발 고정 폴트 (9.1.4)
```
판별 시점: 서브 시작 ~ 임팩트 전 구간
조건: 발목 이동량 > 임계값 (80px, 5프레임 연속)
     OR 발이 코트 표면에서 떨어짐 (도약 감지)
```

## 커스텀 YOLO 학습 계획

### 클래스
| 클래스 | 용도 |
|--------|------|
| `shuttlecock` (0) | 웨이스트·높이·헛치기 폴트 |
| `racket_head` (1) | 샤프트·쉐이크·빽지체·오버헤드 폴트 |

### 수집된 레이블 영상 (서브폴트개발 폴더)
| 영상 | 감지 목표 폴트 |
|------|--------------|
| 정상.mp4 | 없음 (정상 서브 학습) |
| 허리선넘기.mp4 | 웨이스트 / 높이 |
| 선밟기.mp4 | 선밟기 |
| 쉐이크.mp4 | 쉐이크 |
| 풋무브.mp4 | 발이동 |
| 오버헤드.mp4 | 샤프트 / 오버헤드 |
| 빽지체.mp4 | 빽지체 |

### 라벨링 워크플로우
```bash
# 1. 프레임 추출 (5프레임마다 1장)
python extract_frames.py

# 2. Roboflow 업로드 → shuttlecock / racket_head 바운딩박스 라벨링

# 3. YOLOv8 포맷으로 export

# 4. 학습
python train_model.py dataset/data.yaml --epochs 150

# 5. 검증
python train_model.py dataset/data.yaml --val \
    --model runs/train/badminton_v1/weights/best.pt
```

### 목표 성능
- shuttlecock mAP50 ≥ 0.85
- racket_head  mAP50 ≥ 0.80
